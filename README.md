# Personal-fitness-and-macro-tracker
## Overview
The Personal Fitness & Macro Tracker is a commandline Java application designed to help students balance their academic workload with their gym schedules and dietary goals. It allows users to log daily cardiovascular activity, weightlifting focus, and protein intake to ensure they are hitting their fitness targets

## Why This Project?
As a college student, keeping track of daily treadmill minutes, specific muscle group workouts, and daily protein intake (from sources like eggs, paneer, and soya) can get complicated. This tool simplifies the process by keeping a running log of daily achievements in one easy to use interface.

## Prerequisites
To run this project, you will need:
* Java Development Kit installed on your system.
* A terminal or command prompt.

## How to Run the Project
1. Clone this repository to your local machine.
2. Open your terminal and navigate to the folder containing the project.
3. Compile the Java file by running:
   `javac FitnessTracker.java`
4. Run the compiled application by typing:
   `java FitnessTracker`

## How to Use
Upon running the application, you will be greeted with an interactive menu:
* Option 1: Prompts you to enter your daily stats (Date, Treadmill minutes, Workout focus, and Protein in grams)
* Option 2: Displays a formatted history of all the logs you have entered during the current session.
* Option 3: Exits the application.
